# 检索技巧

| 关键字         | Description     |
|----------------|-----------------|
| ""             | 完全匹配        |
|  -xx            | 排除xx          |
|  +xx            | 必须包含xx      |
| - *            | 通配符，谷歌支持 |
| site:x.com     | 限定域名        |
| filetype:doc   | 限定文件类型    |
| inurl:zhihu    | 出现在url中     |
| intitle:zhihu  | 出现在title中   |
| inanchor:zhihu | 出现在锚点中    |
| intext:zhihu   | 出现在正文中    |
